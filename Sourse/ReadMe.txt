----------------------------------------------------
		Description:
----------------------------------------------------
This is the Visual Studio Add-in for boost library tests.
Add-in has GUI that allows make basic operations with boost tests, such as:
- test navigation through the test tree
- test creation (projects with tests, suites, cases)  
- run specific tests and get results


----------------------------------------------------
		System requirements:
----------------------------------------------------
Visual Studio 2010


----------------------------------------------------
		Installation:
----------------------------------------------------
- run installer - SetupTGA.msi.
- the installer create folder c:\Program Files\TGA\ with 2 dll and 2 templates files(for creating new project with tests).
- the installer put add-in file in the "~\User\My Documents\Visual Studio 2010\Addins\" with parameter "<Assembly>c:\Program Files\TGA\TestGuiApp.dll</Assembly>".
- restart your visual studio.
- go to "Tools" and click "TestGuiApp"
